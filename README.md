# MyBestHotel

Система, которая собирает отзывы о гостиницах из разных источников, анализирует их и помогает выбрать лучшую с учетом профиля путешественника.

## Запуск локально

1. Backend: `cd c:\Work\MyBestHotel` → `uvicorn backend.main:app --reload`
2. Frontend: `cd frontend` → `python serve.py`
3. Открыть http://127.0.0.1:5500
